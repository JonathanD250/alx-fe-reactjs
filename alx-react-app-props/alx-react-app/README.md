# ALX Front-End ReactJS Project

This repository contains the first React application for the ALX Front-End specialization. The app is created using Vite with the React template.

## Project Setup

1. Clone the repository:
   git clone <https://github.com/your-username/alx-fe-reactjs>

2. Navigate into the project directory:
   cd alx-fe-reactjs/alx-react-app

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Visit the app in your browser at:
   <http://localhost:5173>

## Tools and Frameworks

- React 18
- Vite
- JavaScript (ES6+)
